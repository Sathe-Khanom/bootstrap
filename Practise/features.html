IT Scholarship Programme
Free: The program is free of cost.
Selection: The program selects candidates based on merit.
Training: The program offers industry-relevant training content.
Assessment: The program assesses trainees' performance through quizzes and evidence-based tests.
Job placement: The program guarantees job placement.
Training providers: The program uses training service providers (TSPs) to provide training infrastructure and instructors.
Training courses: The program offers courses such as web application development, cross platform apps, and Oracle database application development.
Vocational Training Programme
Free: The program is free of cost.
Training: The program offers intensive skill training that lasts six months.
Accommodation: The program provides free accommodation and food facilities.
Maintenance allowance: The program provides a monthly maintenance allowance of Taka 500 for each trainee.
Job placement: The program offers job placement after the successful completion of the training.
Other programs
Madrasah Education Programme: The program introduced Dakhil (Vocational) courses in six Madrasahs.
4-Year Diploma Scholarship Programme: The program offers a 4-year diploma scholarship.
